## How to Get Started

1. **Read** [Meet the Zoo Crew](meet-the-zoo-crew.md) (5 min)
2. **Skim** the [Glossary](glossary.md) for tips on how to phrase requests
3. **Pick an agent** from the table above and open its testing guide
4. **Go to the agent's Discord channel** and start with the Level 1 prompts
5. **Work your way up** through the levels as you get comfortable
6. **Give feedback** -- correct the agents when they're wrong, they learn from it

---

## Quick Tips

- **Be direct.** Skip pleasantries. "Create a tweet about X" > "Hey, could you maybe help me..."
- **Be specific.** More context = better output. Include numbers, links, tone preferences.
- **Iterate.** First output not perfect? Say "make it shorter" or "more degen tone" -- don't start over.
- **They remember.** Agents keep context across messages. Build on previous responses.
- **They have tools.** These agents can browse the web, create files, make presentations, generate images, and talk to each other.
- **Push boundaries.** The advanced capabilities are real -- test them.
